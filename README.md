Консольное приложение для поиска наименьшего свободного IPv4-адреса.

Программа читает список занятых IPv4-адресов из CSV-файла (ipv4.csv) и возвращает первый свободный адрес, начиная от минимального занятого, при этом не возвращает адреса, заканчивающиеся на .0 (сетевые адреса).

Быстрый поиск: все входные адреса конвертируются в uint и хранятся в HashSet — O(n) по времени и O(n) по памяти.

Реализовано по принципам чистой трёхслойной архитектуры (Presentation / BLL / DAL).

Алгоритм поиска находится в: IpFinder1.BLL.Common (класс Finder).

Ввод IP: CSV-файл ipv4.csv (каждый IP на отдельной строке).
